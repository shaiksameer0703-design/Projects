## Major Projects

### VR Augmented Teleoperated Robots for Surgeries 

A teleoperation system that integrates robotics with virtual reality to assist surgeons in performing precise remote operations. The system provides enhanced visualization and control through VR interfaces.

---

### Autonomous UAV Systems for Navigation and Search and Rescue

A drone-based autonomous navigation system designed for search and rescue operations in disaster environments. The system uses path planning algorithms and onboard sensors to locate survivors.

---

### Personal Assistive Robot

A service robot designed to assist users with everyday tasks such as object handling, navigation, and information interaction.

---

### Cognitive Socially Assistive Robot for Autism Therapy

A socially assistive robotic system designed to support autism therapy through multimodal interaction and adaptive behavioral responses.

---

### AC4MPC Control System

Implementation of Actor-Critic Reinforcement Learning for Nonlinear Model Predictive Control. The system demonstrates how reinforcement learning can optimize control performance in dynamic systems.

---

Foundational robotics systems including:

- Maze solving robot

These projects demonstrate fundamental robotics principles including sensor feedback, control algorithms, and mobile robot navigation.
